# A Python wrapper for the IDS database

This wrapper use the IDS data from https://gitlab.chise.org/CHISE/ids

-----

In a python file import IDS

```import ids_py```

<br />
<h2>COMPOSITION</h2>

```ids_py.composition("吃")```
>吃 -> ⿰口乞

```ids_py.composition("吃", flatten=True)```
>吃 -> ⿰口 {乞:⿱ {𠂉:⿰丿一} 乙}

<br />
<h2>STRUCTURE</h2>

```ids_py.structure("畵")```
>畵 -> ⿱⿶
